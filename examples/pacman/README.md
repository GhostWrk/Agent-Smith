# Pac-Man example

Reference implementation for Code Mode web-game tasks. Open `index.html` in a browser.

- Fixed maze layout (no random walls)
- Arrow keys / WASD
- Pellets, score, simple ghost AI
- CSS class `.pacman` matches JS `classList.add('pacman')`
- Template literals use backticks throughout

Verify syntax:

```bash
node --check examples/pacman/script.js
```
